## How to Run
```sh
npm install
npm run dev
```
Then open at localhost:5173
### Compile (if needed)
```sh
npm run build
```

### Dependencies (installed with npm install)
vue3
vue3-google-maps
sass

### Google API Key
Place Google Maps API Key in main.js
```js
app.config.globalProperties.googleMapsKey = 'API_KEY_HERE';
```
